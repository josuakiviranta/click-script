PURPOSE
Manipulate "the most read news"-list that news papers show on their sites.

HOW IT WORKS?
This script open url page repetetive on google chromium browser.
You need to use VPN in order to make this work.

INSTALLING.
This script automatically downloads google chromium on your device.
You also need to install node on your device.

INSTALL PACKAGES:
npm -i

RUN SCRIPT:
node click_script.js <url>
